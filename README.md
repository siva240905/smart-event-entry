# 🎟️ Smart Event Entry System

A modern and responsive Smart Event Entry System developed to simplify event registration and participant management. The system enables organizers to efficiently manage attendees using QR code-based check-ins, real-time attendance tracking, and an intuitive dashboard.

## ✨ Features

- 🎫 Online Event Registration
- 📱 QR Code Generation
- ✅ QR-Based Check-In & Verification
- 👥 Participant Management
- 📊 Real-Time Attendance Tracking
- 📧 Automated Confirmation & Notifications
- 🔍 Search & Filter Participants
- 📈 Admin Dashboard
- 📱 Fully Responsive UI
- ⚡ Fast & Secure Event Management

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- React
- Node.js
- Express.js
- MongoDB
- Vercel (Deployment)

## 🚀 Live Demo

https://smart-event-entry.vercel.app/

## 🎯 Purpose

This project was developed to automate event management by replacing manual registration and attendance processes with a secure QR code-based digital solution. It demonstrates full-stack development, responsive UI design, and real-world event management workflows.
